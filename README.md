# CT5209 Spring Boot WAR CI/CD Pipeline

This project is a Spring Boot WAR application for managing petitions. It was developed for the CT5209 Cloud DevOps continuous assessment.

## Features

- Create petitions
- View petitions
- Search petitions
- View and sign petitions

## Tech Stack

- Java 17
- Spring Boot
- Maven
- Jenkins
- GitHub
- Cloudflare Tunnel

## CI/CD

The application is built and deployed through a Jenkins pipeline triggered automatically from GitHub via webhook.
